# 03 — Certificazioni Cloud

Le certificazioni sono il modo migliore per studiare con un filo logico e farsi notare dai recruiter.

## Contenuto

### [1. Roadmap per Principianti](1-roadmap-principianti.md)
Le 3 certificazioni entry-level con costi, contenuti e a chi sono rivolte.

| Certificazione | Provider | Costo |
|---|---|---|
| AZ-900 Azure Fundamentals | Microsoft | ~165 EUR |
| AWS Cloud Practitioner | Amazon | ~100 USD |
| Google Cloud Digital Leader | Google | ~99 USD |

> Consiglio: per AZ-900 cerca i Virtual Training Days Microsoft — spesso includono voucher gratuito per l'esame.
