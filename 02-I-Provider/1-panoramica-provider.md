# 🏢 I "Big Three": AWS, Azure e Google Cloud

Il mercato del Cloud pubblico è dominato da tre colossi. Ognuno ha centinaia di servizi con nomi diversi che fanno le stesse identiche cose. Ecco una mappa per orientarti.

### 🟧 Amazon Web Services (AWS)
* **La Storia:** È il pioniere. Ha inventato il cloud moderno nel 2006.
* **Quota di mercato:** ~32% (Il leader indiscusso).
* **Punto di forza:** Ha il maggior numero di servizi, la community più grande e la documentazione più vasta. È la scelta di default per le startup.
* **Servizi Base:**
  * Server (IaaS): *EC2 (Elastic Compute Cloud)*
  * Storage File: *S3 (Simple Storage Service)*
  * Database: *RDS (Relational Database Service)*

### 🟦 Microsoft Azure
* **La Storia:** Lanciato nel 2010, ha sfruttato il dominio di Microsoft nelle aziende.
* **Quota di mercato:** ~23% (In forte crescita).
* **Punto di forza:** Se un'azienda usa già Windows, Active Directory, Office 365 o SQL Server, il passaggio ad Azure è naturale e spesso scontato economicamente. È il re del mondo "Enterprise".
* **Servizi Base:**
  * Server (IaaS): *Virtual Machines*
  * Storage File: *Blob Storage*
  * Database: *Azure SQL Database*

### 🟥 Google Cloud Platform (GCP)
* **La Storia:** È arrivato dopo, ma sfrutta la stessa infrastruttura su cui girano Google Search e YouTube.
* **Quota di mercato:** ~10%.
* **Punto di forza:** È il paradiso per i Data Scientist. Imbattibile su Big Data, Machine Learning, Intelligenza Artificiale e container (Google ha inventato Kubernetes!).
* **Servizi Base:**
  * Server (IaaS): *Compute Engine*
  * Storage File: *Cloud Storage*
  * Database: *Cloud SQL*
