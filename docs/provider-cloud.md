Il mercato del cloud pubblico è dominato da tre grandi player, chiamati i **Big Three**. Capire le differenze ti aiuta a scegliere su quale specializzarti.

## Amazon Web Services (AWS)

Lanciato nel 2006, AWS è il cloud provider più maturo e con il catalogo di servizi più ampio — oltre 200 servizi disponibili.

**Punti di forza:**
- Ecosistema vastissimo — c'è un servizio per ogni esigenza
- Community enorme e documentazione eccellente
- Leader nelle startup e nel mondo tech
- Regioni e availability zone ovunque nel mondo

**Punti deboli:**
- Console spesso complessa e dispersiva
- Curva di apprendimento ripida per i principianti
- Costi difficili da stimare in anticipo

**Usato da:** Netflix, Airbnb, Spotify, NASA

**Certificazione entry-level:** AWS Certified Cloud Practitioner (CLF-C02) — ~100$

---

## Microsoft Azure

Il cloud di Microsoft, lanciato nel 2010. Forte integrazione con tutti i prodotti Microsoft — Windows Server, Active Directory, Microsoft 365.

**Punti di forza:**
- Integrazione nativa con ambienti Windows e Microsoft 365
- Ideale per aziende enterprise già su ecosistema Microsoft
- Azure Active Directory — standard de facto per l'identità aziendale
- Forte presenza in Europa, ottima per compliance GDPR

**Punti deboli:**
- Alcuni servizi meno maturi rispetto ad AWS
- Naming dei servizi spesso confuso (continua a cambiare)

**Usato da:** BMW, Volkswagen, Deutsche Bank, molte PA europee

**Certificazione entry-level:** AZ-900 Azure Fundamentals — ~165€

---

## Google Cloud Platform (GCP)

Il cloud di Google, forte sulle stesse tecnologie usate internamente da Google — Kubernetes nasce qui, BigQuery è ancora imbattuto per analytics.

**Punti di forza:**
- Kubernetes: Google ha creato Kubernetes e GKE è la migliore implementazione managed
- BigQuery: data warehouse serverless eccezionale per analisi di grandi dataset
- AI e Machine Learning: Vertex AI, TensorFlow, modelli Gemini
- Rete globale di Google — latenza bassissima

**Punti deboli:**
- Quota di mercato più bassa = meno richiesto dai recruiter
- Meno servizi rispetto ad AWS
- Google ha storicamente "abbandonato" prodotti cloud

**Usato da:** Spotify (anche su GCP), Twitter, Snap, PayPal

**Certificazione entry-level:** Google Cloud Digital Leader — ~99$

## Quale scegliere per iniziare?

**Vuoi lavorare in aziende strutturate o in Italia?** → Azure — l'ecosistema Microsoft domina le PMI e le grandi aziende italiane.

**Vuoi lavorare in startup o aziende tech?** → AWS — è il gold standard del settore, la certificazione più richiesta.

**Sei appassionato di ML e dati?** → GCP — Vertex AI e BigQuery sono strumenti eccezionali.

---

> Non devi scegliere per sempre — molti professionisti hanno certificazioni su più provider. Inizia da uno, padroneggiali i fondamenti, poi espandi.
