# Roadmap Certificazioni Cloud

Le certificazioni non ti rendono automaticamente un senior, ma sono il modo migliore per studiare con un filo logico e dimostrare le tue competenze ai recruiter.

---

## Come funzionano le certificazioni cloud

Ogni provider ha una propria gerarchia di certificazioni, organizzate per livello e specializzazione.

```
Foundational (Entry-Level)    → Concetti base, nessun prerequisito
    │
    ▼
Associate                     → Ruoli specifici, 6-12 mesi esperienza consigliata
    │
    ▼
Professional / Expert         → Architetture avanzate, esperienza richiesta
    │
    ▼
Specialty                     → Verticali specifici (ML, Security, Database...)
```

Questa guida si concentra sul livello **Foundational** — il punto di partenza ideale.

## Le 3 Certificazioni Entry-Level

### AZ-900 — Azure Fundamentals (Microsoft)

**A chi è rivolta:** Chi punta ad ambienti aziendali strutturati, specialmente in Italia dove Azure domina il mercato enterprise.

**Cosa impari:**
- Concetti fondamentali del cloud computing
- Servizi core di Azure (compute, storage, networking, database)
- Sicurezza, privacy e compliance
- Modelli di pricing e SLA

**Dettagli esame:**
- Costo: ~165€
- Durata: 45 minuti
- Domande: 40-60 (scelta multipla, scenari)
- Punteggio minimo: 700/1000
- Validità: a vita (non scade)

**Consiglio pratico:** Microsoft organizza periodicamente i **Virtual Training Days** — webinar gratuiti di 2 giorni al termine dei quali viene spesso offerto un voucher per sostenere l'esame senza costo. Cerca "AZ-900 Virtual Training Day" su eventbrite.com o sul sito Microsoft.

---

### AWS Certified Cloud Practitioner — CLF-C02 (Amazon)

**A chi è rivolta:** Chi vuole entrare nell'ecosistema AWS, la certificazione più richiesta nel mercato tech globale.

**Cosa impari:**
- Architettura AWS e servizi principali (EC2, S3, RDS, Lambda)
- Sicurezza e shared responsibility model
- Modelli di fatturazione e ottimizzazione dei costi
- Well-Architected Framework

**Dettagli esame:**
- Costo: ~100$
- Durata: 90 minuti
- Domande: 65 (scelta multipla e risposta multipla)
- Punteggio minimo: 700/1000
- Validità: 3 anni

**Consiglio pratico:** AWS offre un piano gratuito (Free Tier) per 12 mesi — usalo per esercitarti con i servizi reali mentre studi.

---

### Google Cloud Digital Leader (Google)

**A chi è rivolta:** Chi vuole comprendere GCP con un focus su trasformazione digitale, dati e AI.

**Cosa impari:**
- Fondamenti del cloud e della trasformazione digitale
- Prodotti e servizi Google Cloud
- Dati e intelligenza artificiale su GCP
- Infrastruttura e sicurezza cloud

**Dettagli esame:**
- Costo: ~99$
- Durata: 90 minuti
- Domande: 50-60
- Punteggio minimo: 70%
- Validità: 3 anni

## Ordine consigliato

```
Parti da qui:
    AZ-900 (gratuito con Virtual Training Day)
        │
        ▼
    Valuta il tuo percorso:
        │
        ├── Lavoro in Italia / aziende enterprise
        │   └── AZ-104 (Azure Administrator)
        │
        ├── Startup / mondo tech globale
        │   └── AWS Solutions Architect Associate
        │
        └── Dati / Machine Learning
            └── Google Cloud Digital Leader → DP-100
```
